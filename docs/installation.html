Installation

  1. Clone this GitHub repo

          
        {
        git clone https://github.com/wavrusawi-collab/easydesk.git
        }

  2. In the directory of the cloned repo, make a python3 venv

  3. Install PyQt6 and PyQt6-WebEngine

          
        {
        pip install PyQt6 PyQt6-WebEngine
        }

  4. Run the script

          
        {
        python main.py
        }
NOTICE: If you have a pre-built version, launch the file or app.
