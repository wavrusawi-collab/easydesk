## Installation
1. Clone this repo using `git clone https://github.com/wavrusawi-collab/easydesk.git`
2. Move to the directory of the cloned repo
3. Add a python3 venv
4. Install PyQt6 and PyQt6-WebEngine by using `pip install PyQt6 PyQt6-WebEngine`
5. Run the script with '''python main.py'''
***NOTICE: If you have a pre-built version, just double click the file or go into the app launcher.**
